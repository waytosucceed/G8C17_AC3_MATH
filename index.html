Multiplication of Exponents with the Same Base but Different Powers

When you're multiplying two numbers that have the same base but different powers, there's another simple rule:
Rule: When you multiply exponents with the same base, you add the powers.

In mathematical terms: am × an = am+n

Example:
Let's take a base of 3:
32 × 34 = 9 × 81 = 729
Using the exponent rule, we add the powers:
32 × 34 = 32+4 = 36
Now, calculate 36:
36 =3×3×3×3×3×3 = 729

Summary of the Rules:

  1. Same Base, One Power Inside Another (Multiple Exponents):(am)n = am.n
  2. Same Base, Different Powers Being Multiplied:am × an = am+n
